Final Year Project

All In One Visual Tracking System

Project By,

Arun Vignesh S, Gracy S, Harinaath K

In this project we worked on bringing together all the available features of visual tracking such as Image tracking and face tracking. The best best part of this project is that it can track both faces and images and have the ability to bring in some web augmented reality using the help MINDAR an opensource augmented reality library.
